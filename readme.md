### Overview of different modules
This project contains following modules
1) `coreKotlin` module.
2) `kotlinJava` module.
3) `kotlinPlayground` module.
4) `practiceSolutions` module.

- `coreKotlin`: This module contains all the programs, which are demonstrations of concepts used in Kotlin, such
language features, lambdas, data types, use of OOJ, lists, and other core concepts that are used in Kotlin.
- `kotlinJava`: This module contains all the programs that show how easily Java and Kotlin code can work together. It has several programs that show how we can achieve Java-Kotlin interoperability.
- `kotlinPlayground`: An IDE-based Kotlin playground that you can use to experiment on different features of Kotlin and
  Run these programs to test the output using this module.

### Practice problem statements
1) Create a function that takes two numbers as arguments and returns their sum.
2) Write a function that takes an integer minutes and converts it to seconds.
3) Create a function that takes a number as an argument, increments the number by +1 and returns the
   result.
4) Write a function that takes the base and height of a triangle and return its area.
5) Create a function that takes the age in years and returns the age in days.
6) Create a function that takes an array containing only numbers and return the first element.
